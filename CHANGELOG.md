# Changelog

## v1.1.0

- add private key authentication, strict host key verification, and command timeouts
- add remote file operations including mkdir, remove, chmod, move, and atomic upload
- add JSON deployment plans for repeatable multi-step remote workflows
- add one-way and basic bidirectional sync with include/exclude filters and dry-run support
- improve large file handling with streaming upload and download
- add tests, example plans, and expanded documentation for deployment usage
